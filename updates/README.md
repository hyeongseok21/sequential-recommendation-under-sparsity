# Update Logs

This directory stores chronological experiment and packaging notes.

## Structure

- `0.local-setup/`
  - environment bring-up, runtime compatibility, and seed stabilization
- `1.sasrec-vs-meta-embedding/`
  - early SASRec baseline and metadata-embedding comparisons
- `2.sasrec-vs-meta-embedding-vs-difsr/`
  - DIF-SR implementation, metadata fusion, and backbone-level modeling experiments
- `3.framework-and-closure/`
  - checkpoint policy, evaluation policy, framework documentation, and closure packaging work
- `4.portfolio-closure/`
  - final phase-agent closure runs and packaging execution logs
